Project
This is our awesome project, which is a responsive site designed using front-end tech like HTML CSS js. Here, the user is supposed to upload the images via direct capturing or browse the image and to make this interactive with machine learning we used Django.

The images are processed by stages internally like data pre-processing, featured engineering, it pre-processes those images fetching relevant datasets and checks suitable to this model is appropriately match with the given image.

Techniques
By using this structured algorithmic technique the model is trained with the selected algorithm, it will predict the skin tone of the face present in the image and classify it into fairly mild dark. Finally the result is displayed on the web page.
